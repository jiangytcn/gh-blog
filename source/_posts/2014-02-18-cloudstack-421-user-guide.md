### Adding nic to a vm
- **Be sure that the network to be added had a VR running**
- **The network doesn't add before**
- **Shutdown VM**
<pre><code>Click INSTANCES -->> VM -->> Details -->> NICS -->> Add network to VM  -->> select network -->> OK
</code></pre>
